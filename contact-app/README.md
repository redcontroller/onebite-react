# 주요 기능 및 특징

**특징**

- React Web Application
- Chance 라이브러리를 통한 더미 연락처 생성
- useReducer와 Context를 이용한 상태관리
- React 렌더링 최적화
  - useMemo, React.memo, useCallback 활용
